# Power BI Dashboard Project

## Training Project

This dashboard was developed as a hands-on training project to strengthen my skills in data modeling, DAX, and business-focused analytics using real-world e-commerce scenarios.

---

## Project Overview

As a Data Analyst, I always ask one important question:

How can data move from numbers on a screen to real strategic impact?

To answer that, I built an interactive **E-Commerce Sales Analysis Dashboard in Power BI** designed to transform raw sales data into actionable business insights.

---

## Key Performance Indicators (KPIs)

- Total Sales  
- Total Orders  
- Total Customers  
- Total Profit  
- Average Order Value  

The main goal was not only to visualize data but to identify performance drivers and uncover hidden business opportunities.

---

## Dashboard Features

- Track sales trends across months and years  
- Identify the most profitable product categories  
- Analyze geographic performance by state and city  
- Compare Sales vs Profit to evaluate business efficiency  
- Perform advanced drill-down analysis by Category, Year, State, and Seller ID  

---

## Technical Implementation

- Built a structured data model for multi-dimensional analysis  
- Created DAX measures for key business metrics  
- Developed interactive dashboards with dynamic filters  
- Used Azure Maps for geographic visualization  
- Implemented Drill-Down analysis for deeper insights  

---

## Key Insight

High sales do not always mean high profitability.

Some product categories generate high revenue but low margins, which highlights the importance of profit-driven decision making rather than focusing on revenue alone.

---

## Conclusion

Data is not just numbers.

It is a strategic asset that helps businesses grow, optimize performance, and make smarter decisions.
